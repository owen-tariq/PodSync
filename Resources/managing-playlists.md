# Managing Playlists

Playlists help you organize your music into collections for different moods, activities, or projects. PodCenter supports playlists in both your library and on your iPod.

## Creating a Playlist

### In Your Library

Click the **+** button next to the "Playlists" header in the sidebar. A new playlist called "Untitled Playlist" is created and you can immediately type a name for it.

**Alternative methods:**
- **Menu Bar:** Edit → New Playlist... (⌘⇧N) — this also adds any selected tracks
- **Right-click tracks:** Select tracks, right-click → Add to Playlist → New Playlist...

### On Your iPod

Click the **+** button next to the "Playlists" header under your iPod in the sidebar. The playlist is created directly on the device.

> **Note:** This only works with writable iPod models. Read-only devices won't show the + button.

> **Important:** Empty playlists won't appear on your iPod until they contain at least one track. This is a limitation of the iPod database format. Add songs to the playlist to make it visible on the device.

## Adding Tracks to a Playlist

### Drag and Drop

1. **Select tracks** in your library (⌘-click for multiple)
2. **Drag** them to a playlist in the sidebar
3. **Drop** when the playlist highlights

### Context Menu

1. **Select tracks** you want to add
2. **Right-click** on the selection
3. Choose **Add to Playlist** → select the target playlist

### Edit Menu

With tracks selected, go to Edit → Add to Playlist and choose your playlist.

## Reordering Tracks

To change the order of tracks in a playlist:

1. **Open** the playlist by clicking it in the sidebar
2. **Click and hold** on a track row
3. **Drag** it to a new position
4. **Drop** to place it

After reordering, **Apply** and **Discard** buttons appear in the playlist header:
- Click **Apply** to save the new order
- Click **Discard** to revert to the original order

> **Note:** Changes are not saved automatically — you must click Apply to confirm.

## Removing Tracks from a Playlist

1. **Open** the playlist
2. **Select** the tracks you want to remove
3. Press **Delete** or right-click and choose **Remove from Playlist**

> **Note:** This only removes the track from the playlist — the audio file stays in your library.

## Syncing Playlists to iPod

1. **Right-click** on a playlist in your library sidebar
2. Select **Sync to [Device Name]** — the sync starts immediately

If some of the playlist's tracks are already on the device, a duplicate-resolution dialog appears so you can choose what to do with them.

If you have multiple iPods connected, you'll see a submenu to choose which device.

## Renaming a Playlist

- **Right-click** on the playlist → **Rename...** (or press Return)
- Type the new name and press Return to confirm

## Duplicating a Playlist

Right-click on the playlist → **Duplicate** (⌘D) to create a copy with the same tracks.

## Deleting a Playlist

1. **Right-click** on the playlist → **Delete '[Name]'**
2. Confirm in the dialog

You can also select the playlist and press **Delete** (when no tracks are selected).

> **Note:** Deleting a playlist can be undone with Edit → Undo (⌘Z). Either way, the songs in the playlist are never removed from your library.

## Keyboard Shortcuts

### Global Shortcuts

| Action | Shortcut | How to Use |
|--------|----------|------------|
| New Playlist (with selected tracks) | ⌘⇧N | Works globally when tracks are selected |

### When Playlist Is Selected in Sidebar

These shortcuts work when you have a playlist highlighted in the sidebar and no tracks are selected:

| Action | Shortcut | Notes |
|--------|----------|-------|
| Delete Playlist | Delete | Shows confirmation dialog |
| Duplicate Playlist | ⌘D | Library playlists only |
| Rename Playlist | Return | Enters inline editing mode |

### When Tracks Are Selected in a Playlist

| Action | Shortcut |
|--------|----------|
| Remove Tracks from Playlist | Delete |

> **Tip:** The Delete key behavior depends on context: if tracks are selected, it removes them from the playlist. If no tracks are selected but a playlist is highlighted in the sidebar, it deletes the playlist itself.
